# Food Store

Sistema web de gestión de pedidos de comida desarrollado para la materia Programación III de la Tecnicatura Universitaria en Programación.

La aplicación permite visualizar un catálogo de productos, buscar y filtrar por categorías, agregar productos al carrito y mantener la persistencia de los datos utilizando `localStorage`.

## Tecnologías utilizadas

- HTML5
- CSS3
- JavaScript
- TypeScript
- Vite

## Funcionalidades implementadas

- Catálogo dinámico de productos
- Búsqueda de productos por nombre
- Filtrado por categorías
- Carrito de compras
- Persistencia del carrito con `localStorage`
- Modificación de cantidades
- Eliminación de productos del carrito
- Cálculo automático del total de compra

## Cómo ejecutar el proyecto

### 1. Descargar o clonar el repositorio

```bash
git clone https://github.com/GonzaloTozzi/Evaluacion-Programacion-III-Tozzi-Gonzalo
```

### 2. Ingresar a la carpeta del proyecto

Abrir una terminal dentro de la carpeta raíz del proyecto.

### 3. Instalar dependencias

```bash
pnpm install
```

### 4. Levantar el servidor de desarrollo

```bash
pnpm dev
```

El proyecto se ejecutará en:

```bash
http://localhost:5173
```

## Build de producción

Para generar el build del proyecto:

```bash
pnpm build
```
